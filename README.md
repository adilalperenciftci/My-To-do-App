# TodoMatic Uygulaması

Bu proje, kullanıcıların günlük görevlerini yönetmelerine yardımcı olan basit, kullanıcı dostu bir web uygulamasıdır. React kullanılarak geliştirilmiştir ve modern bir kullanıcı arayüzüne sahiptir.

## Özellikler

- Görev ekleme
- Görevleri tamamlandı olarak işaretleme
- Görev düzenleme
- Görev silme
- Görevleri filtreleme (Tümü, Aktif, Tamamlanan)

## Teknolojiler

- React
- CSS3
- HTML5
- [nanoid](https://www.npmjs.com/package/nanoid) (ID oluşturma için)

## Kurulum

Bu projeyi yerel olarak çalıştırmak için aşağıdaki adımları izleyin:

1. Repoyu klonlayın:

   ```bash
   git clone https://github.com/[kullanıcı-adınız]/todomatic.git
   cd todomatic
   ```

2. Gerekli paketleri yükleyin:

   ```bash
   npm install
   ```

3. Uygulamayı başlatın:

   ```bash
   npm start
   ```

   Bu komut, tarayıcınızda `http://localhost:3000` adresinde uygulamayı başlatır.
